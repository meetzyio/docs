# Import Contacts

Meetzy allows you to import contacts in bulk from Excel files (.xlsx, .xls, .xlsm) or directly from Google Sheets. This enables you to quickly populate your contact database for campaigns with advanced filtering and AI-powered search capabilities.

## Import Methods

| Method | Description | Requirements |
|--------|-------------|--------------|
| 📊 Excel | Upload .xlsx, .xls, or .xlsm files | File up to 10MB |
| 📗 Google Sheets | Connect directly to your spreadsheets | Google account integration |

## Import Wizard

The import process follows three steps:

<Steps>
  <Step title="Upload / Select">
    Choose your source file or spreadsheet.
  </Step>
  <Step title="Configure">
    Map columns and set import options with advanced filters.
  </Step>
  <Step title="Import">
    Process contacts and view results.
  </Step>
</Steps>

## Excel Import

### Uploading a File

1. Navigate to **CRM** → **Import Contacts**
2. Select **Import from Excel**
3. Drag and drop your file or click to browse
4. Supported formats: `.xlsx`, `.xls`, `.xlsm`
5. Maximum file size: 10MB

<Info>
Drag and drop works directly on the upload zone. The border turns green when a valid file is detected.
</Info>

### File Validation

Once uploaded, the system analyzes your file:

| Analysis | Description |
|----------|-------------|
| Total Rows | Number of data rows detected |
| Valid Phones | Rows with valid phone numbers |
| Duplicates | Duplicate phone numbers found |
| No Phone | Rows missing phone numbers |

<Warning>
Rows without phone numbers will be skipped during import. Ensure your phone column is properly formatted.
</Warning>

## Google Sheets Import

### Connecting Google Sheets

If not connected:

1. Click **Connect Google Sheets**
2. Authorize with your Google account
3. Grant access to your spreadsheets

### Selecting a Spreadsheet

1. Browse your available spreadsheets
2. Use the search bar to find specific files
3. Click on a spreadsheet to select it
4. The system validates the sheet structure

## Advanced Filters & Search

### AI-Powered Search

Use natural language to find specific contacts:

<Steps>
  <Step title="Open AI Search">
    Click the AI search icon in the search bar.
  </Step>
  <Step title="Enter Query">
    Type your search in natural language, e.g., "Find all contacts from California with more than 5 calls"
  </Step>
  <Step title="Review Results">
    AI will process your query and display matching contacts.
  </Step>
</Steps>

<Info>
AI search works with contact data, call history, location information, and custom fields to provide intelligent results.
</Info>

### Advanced Filtering

Access advanced filters to narrow down your import data:

| Filter Type | Options | Description |
|-------------|---------|-------------|
| **Status** | Active, Inactive, All | Filter by contact status |
| **Call Range** | Min/Max calls | Filter by number of previous calls |
| **Date Range** | From/To dates | Filter by creation or last contact date |
| **Location** | City, Country | Filter by geographic location |
| **Contact Info** | Has Email | Filter contacts with/without email |

### Smart Filter Combinations

Combine multiple filters for precise targeting:

```
Status: Active + 
Min Calls: 3 + 
Has Email: Yes + 
City: Madrid
```

This finds active contacts from Madrid with at least 3 calls and email addresses.

## Column Mapping

### Required Fields

| Field | Description | Required |
|-------|-------------|----------|
| Phone | Contact phone number | ✅ Yes |
| Name | Contact name | No |
| Email | Contact email | No |

### Enhanced Mapping Options

1. **Standard Fields**: Phone, Name, Email
2. **Location Fields**: City, State, Country, Postal Code
3. **Business Fields**: Company, Job Title
4. **Marketing Fields**: UTM Source, Medium, Campaign
5. **Custom Fields**: Any additional columns from your file

### Smart Column Detection

The system automatically suggests column mappings based on:
- Column headers
- Data patterns
- Common naming conventions

<Info>
The system recognizes headers like "First Name", "fname", "given_name" and maps them appropriately.
</Info>

### Custom Fields

Add additional columns as custom fields:

1. View available unmapped columns
2. Click on a column name to include it
3. Custom fields are stored with the contact
4. Use them in agent prompts with input parameters

<Warning>
Columns already mapped to standard fields (Phone, Name, Email) cannot be added as custom fields.
</Warning>

### Enhanced Preview

Before importing, review your data with advanced preview features:

- **Data Quality Indicators**: Shows data completeness and quality scores
- **Duplicate Detection**: Highlights potential duplicate contacts
- **Format Validation**: Verifies phone numbers and email formats
- **Geographic Validation**: Validates city/country combinations

### Data Quality Metrics

| Metric | Description | Action |
|--------|-------------|---------|
| ✅ Valid | Properly formatted data | Ready to import |
| ⚠️ Warning | Potentially incorrect data | Review recommended |
| ❌ Error | Invalid data format | Must fix before import |

## AI-Enhanced Data Enrichment

### Automatic Data Enhancement

During import, the system can automatically:

1. **Format phone numbers** with proper country codes
2. **Standardize addresses** using geographic databases  
3. **Detect duplicates** using fuzzy matching algorithms
4. **Enrich contact data** with additional information

### Intelligent Duplicate Handling

Choose how to handle duplicates:

| Option | Description | Recommendation |
|--------|-------------|----------------|
| Skip | Skip duplicate entries | Best for clean data |
| Update | Update existing contacts | Good for data refresh |
| Create New | Create duplicate entries | Use with caution |

## Segment Assignment

All imported contacts are automatically added to a segment:

1. Enter a **Segment Name**
2. This creates a new segment with your imports
3. Use segments for targeted campaigns
4. Apply advanced filters to segment creation

<Info>
Use descriptive segment names like "Q1 2024 Leads - Madrid Tech Companies" for easy identification and targeting.
</Info>

### Smart Segment Suggestions

Based on your data, the system suggests segment names:
- **By Location**: "Madrid Contacts", "Spain Prospects"  
- **By Source**: "LinkedIn Import March 2024"
- **By Industry**: "Tech Companies", "Real Estate Leads"

## Import Process

Once configuration is complete:

1. Click **Start Import**
2. Real-time progress with advanced metrics
3. Live stats display:
   - New contacts created
   - Existing contacts updated
   - Data enrichment applied
   - Rows skipped with reasons

### Enhanced Import Duration

Processing includes:
- **Data validation** and formatting
- **Duplicate detection** and resolution  
- **AI-powered data enrichment**
- **Advanced segmentation**

Large imports (10,000+ rows) may take several minutes depending on enrichment settings.
