## El Nuevo Flujo Prompt-First

Meetzy ahora ofrece una experiencia mejorada para crear agentes. Simplemente describe qué debe hacer tu agente y nuestra IA construirá la configuración inicial por ti.

<Info>
  **¿Nuevo en Meetzy?** El flujo prompt-first es la manera más rápida de empezar. Para usuarios avanzados, también puedes crear un agente desde cero.
</Info>

## Requisitos Previos

Antes de crear un agente, asegúrate de tener:

- Una cuenta activa de Meetzy con acceso al dashboard
- Al menos un número de teléfono disponible en tu cuenta
- Una idea clara de tu caso de uso (ventas, soporte, etc.)

## Crear un Agente con Prompt-First

<Steps>
  <Step title="Accede al Editor Avanzado">
    Desde el dashboard, haz clic en "Create Agent" o navega directamente al Editor Avanzado.
  </Step>
  
  <Step title="Describe tu Agente">
    En la pantalla inicial, encontrarás un campo de texto con fondo animado. Describe qué debe hacer tu agente:
    
    ```
    Un agente que responde llamadas de atención al cliente para una clínica dental, 
    agenda citas y transfiere casos urgentes a la recepcionista...
    ```
  </Step>
  
  <Step title="Genera la Configuración">
    Presiona **Enter** o haz clic en el botón de envío. Meetzy generará automáticamente:
    - Prompt de sistema optimizado
    - Configuración básica recomendada
    - Saludos personalizados
    - Parámetros de entrada sugeridos
  </Step>
  
  <Step title="Refina y Personaliza">
    Revisa la configuración generada y ajusta según tus necesidades específicas usando las herramientas del Editor Avanzado.
  </Step>
</Steps>

<Warning>
  Si prefieres empezar desde cero, puedes hacer clic en "or start from scratch" en la pantalla inicial.
</Warning>

## El Editor Avanzado

El Editor Avanzado es una interfaz de tres paneles para configurar tu agente:

| Panel | Descripción |
|-------|-------------|
| **Barra de Iconos** (izquierda) | Navegación entre secciones de configuración |
| **Editor Central** | Área principal de edición para el prompt de sistema y ajustes |
| **Panel Derecho** | Playground para pruebas y asistente IA |

### Acciones de la Barra de Herramientas

| Botón | Descripción |
|-------|-------------|
| ← **Back** | Volver a la lista de agentes |
| 🏷️ **Version Badge** | Versión actual (ej., v.1, v.2) |
| ✓ **Check** | Validar configuración |
| ⬇️ **Pull** | Sincronizar con la última versión |
| 🚀 **Publish** | Desplegar a producción |

## Configuración del Agente

### 1. Settings (⚙️)

La configuración básica incluye:

#### Nombre del Agente
Identificador interno descriptivo:
- "Agente de Cualificación de Ventas"
- "Atención al Cliente - Facturación"
- "Recordatorio de Citas"

#### Modelo de Lenguaje (LLM)
Selecciona el modelo de IA que potenciará las conversaciones:

<Tabs>
  <Tab title="Anthropic (Recomendado)">
    **Claude Haiku 4.5** - Ultra rápido, excelente para tool calling
  </Tab>
  <Tab title="OpenAI">
    - **GPT-4o** - Equilibrio entre velocidad y calidad
    - **GPT-4.1** - Último modelo completo
    - **GPT-4.1 Mini** - Más rápido y económico
  </Tab>
  <Tab title="Google">
    - **Gemini 2.5 Flash** - Rápido y capaz
    - **Gemini 1.5 Flash** - Ultra rápido, buen equilibrio
    - **Gemini 1.5 Pro** - Más capaz, mayor latencia
  </Tab>
</Tabs>

#### Número de Teléfono

<Info>
  Los números de teléfono son recursos exclusivos. Cada número solo puede estar asignado a un agente a la vez.
</Info>

Selecciona un número disponible o solicita nuevos números mediante "Request more numbers".

#### Componentes Foundation
Mejoras opcionales para tu agente:
- **Safety Guardrails** - Previene respuestas inapropiadas
- **Conversational Aspects** - Mejora el flujo natural
- **Agent Work Experience** - Comportamientos pre-entrenados

### 2. Greetings (👋)

Configura mensajes de bienvenida personalizados:

#### Saludo Entrante
```
¡Hola! Gracias por llamar a {{company_name}}. ¿En qué puedo ayudarte hoy?
```

#### Saludo Saliente
```
Hola {{customer_name}}, soy {{agent_name}} de {{company_name}}. 
Te llamo respecto a {{reason}}. ¿Es buen momento para hablar?
```

### 3. Input Parameters (📄)

Define variables dinámicas para personalización:

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `phone` | String | Teléfono del cliente |
| `name` | String | Nombre del cliente |
| `email` | String | Email del cliente |
| `company` | String | Empresa del cliente |
| `lead_source` | String | Origen del lead |

### 4. Knowledge Base (📚)

Sube documentos de referencia:
- FAQs de productos
- Información de precios
- Políticas de la empresa
- Objeciones comunes y respuestas

<Info>
  El agente busca en la knowledge base contextualmente durante las conversaciones, recuperando información relevante cuando es necesaria.
</Info>

### 5. Actions & Tools (🔧)

Configura integraciones y acciones:

| Acción | Descripción |
|--------|-------------|
| **Transfer Call** | Derivar a agente humano |
| **Send SMS** | Enviar mensaje de texto |
| **Send Email** | Enviar email al cliente |
| **Calendar Booking** | Agendar citas |
| **CRM Update** | Actualizar registros |
| **Custom Webhook** | Llamar APIs externas |

### 6. Evaluations (📊)

Crea criterios de éxito automatizados:

```
Escenario: El cliente pregunta sobre precios
Esperado: El agente NO debe discutir precios específicos
Esperado: El agente debe ofrecer conectar con el equipo de ventas
```

### 7. Output Settings (📥)

Define qué información debe capturar el agente:

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `qualified` | Boolean | ¿El lead está cualificado? |
| `budget_range` | String | Rango de presupuesto |
| `timeline` | String | Cronograma de compra |
| `pain_points` | Array | Desafíos del cliente |

## Prompt de Sistema

El editor central contiene el **Prompt de Sistema** - las instrucciones principales que definen el comportamiento de tu agente.

### Estructura Recomendada

```markdown
Eres un agente amigable de cualificación de ventas para Acme Corp.

## Objetivo
Cualificar leads entendiendo sus necesidades y presupuesto,
luego agendar una demo con un representante de ventas.

## Directrices
- Sé conversacional y amigable
- Pregunta sobre sus desafíos actuales
- Entiende su cronograma y presupuesto
- Si está cualificado, ofrece agendar una demo

## Restricciones
- Nunca discutas precios directamente
- No hagas promesas sobre características
- Transfiere a un humano si el cliente está molesto
```

## Panel Derecho: Herramientas de Desarrollo

### Meetzy AI (✨)
Tu asistente personal para construir agentes. Puede:
- Generar configuraciones automáticamente
- Optimizar prompts existentes
- Sugerir mejoras basadas en mejores prácticas
- Responder preguntas sobre configuración

### Playground (▶️)
Simula conversaciones para probar tu agente antes de publicar.

### Tests (✅)
Ejecuta pruebas automatizadas para validar el comportamiento del agente.

### Changes (📝)
Revisa y gestiona cambios pendientes con un sistema similar a Git.

## Publicar Tu Agente

<Steps>
  <Step title="Validar">
    Haz clic en **Check** para validar tu configuración. Corrige cualquier error.
  </Step>
  <Step title="Probar">
    Usa el Playground para simular conversaciones y verificar el comportamiento.
  </Step>
  <Step title="Hacer llamadas de prueba">
    Usa "Test Agent" para hacer llamadas telefónicas reales de prueba.
  </Step>
  <Step title="Publicar">
    Haz clic en **Publish** para desplegar tu agente a producción.
  </Step>
</Steps>

<Warning>
  Publicar crea una nueva versión. La versión anterior permanece disponible para rollback si es necesario.
</Warning>
