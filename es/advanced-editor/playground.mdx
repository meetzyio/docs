## ¿Qué es el Playground?

El Playground es un entorno de pruebas en el Editor Avanzado donde puedes interactuar con tu agente en tiempo real. Te permite simular conversaciones, probar diferentes escenarios y validar el comportamiento de tu agente antes de desplegar a producción.

<Info>
Probar en el Playground ahorra tiempo y dinero al detectar problemas antes de que afecten a llamadas reales de clientes.
</Info>

## Nuevas Funcionalidades del Playground

### Reconocimiento de Voz Integrado

El Playground ahora incluye reconocimiento de voz automático que convierte tu habla en texto de manera instantánea:

<Tabs>
  <Tab title="Entrada por Voz">
    **Funcionalidades de Voz**
    - Reconocimiento automático de voz en tiempo real
    - Transcripción instantánea a texto
    - Soporte para múltiples idiomas
    - Integración perfecta con el flujo de conversación
  </Tab>
  <Tab title="Configuración de Idioma">
    **Selección de Idioma**
    - Configuración automática basada en el idioma del agente
    - Opción de cambio manual durante la conversación
    - Detección automática del idioma hablado
    - Soporte para dialectos regionales
  </Tab>
</Tabs>

### Estadísticas de Caracteres y Tokens

El editor ahora proporciona información detallada sobre el uso de texto en tiempo real:

| Métrica | Descripción | Umbral |
|---------|-------------|---------|
| 🟢 Óptimo | Menos de 25,000 caracteres | Precisión máxima |
| 🟡 Advertencia | 25,000 - 40,000 caracteres | Posible pérdida de consistencia |
| 🔴 Crítico | Más de 40,000 caracteres | Alto riesgo de inconsistencias |

<Warning>
Las instrucciones muy largas pueden hacer que el agente pierda coherencia en partes específicas. Considera simplificar o dividir el contenido.
</Warning>

### Atajos de Teclado Mejorados

El Playground incluye nuevos atajos de teclado para una experiencia más eficiente:

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| `Ctrl/Cmd + Enter` | Enviar mensaje | Envía el mensaje actual |
| `Ctrl/Cmd + K` | Limpiar conversación | Borra el historial de chat |
| `Ctrl/Cmd + R` | Reiniciar agente | Reinicia la sesión del agente |
| `Espacio` | Activar/Desactivar micrófono | Control rápido de grabación |
| `Escape` | Cancelar grabación | Cancela la grabación de voz activa |

## Accediendo al Playground

El Playground está ubicado en el panel derecho del Editor Avanzado. Haz clic en la pestaña "Playground" para acceder, y observa el estado de sincronización en la barra superior para asegurar que estás probando la versión más reciente.

## Tipos de Llamada

Puedes simular dos tipos de llamadas:

| Tipo | Descripción | Caso de Uso |
|------|-------------|----------|
| 📞 Entrante | Cliente llamando a tu agente | Probar flujo de bienvenida, escenarios de soporte |
| 📤 Saliente | Agente llamando a un cliente | Probar introducciones, scripts de ventas |

<Tip>
Cambia entre tipos de llamada para probar ambos saludos y asegurar que tu agente maneja cada escenario correctamente.
</Tip>

## Enviando Mensajes

### Entrada de Texto Mejorada

Escribe mensajes en el campo de entrada con nuevas funcionalidades:
- **Auto-completado inteligente** basado en conversaciones anteriores
- **Sugerencias contextuales** que aparecen mientras escribes
- **Contador de caracteres** en tiempo real
- **Vista previa del mensaje** antes de enviar

### Entrada de Voz con Reconocimiento Automático

<Steps>
  <Step title="Activar Reconocimiento">
    Haz clic en el icono del micrófono o presiona la barra espaciadora para comenzar a grabar.
  </Step>
  <Step title="Hablar Claramente">
    Habla de forma natural. El sistema detectará automáticamente tu idioma y convertirá tu voz a texto.
  </Step>
  <Step title="Revisar Transcripción">
    La transcripción aparece en tiempo real. Puedes editarla antes de enviarla.
  </Step>
  <Step title="Enviar o Continuar">
    Envía el mensaje transcrito o continúa hablando para añadir más contenido.
  </Step>
</Steps>

<Info>
El reconocimiento de voz se optimiza automáticamente según el idioma principal de tu agente y puede detectar cambios de idioma durante la conversación.
</Info>

## Vista de Conversación Mejorada

El Playground muestra mensajes cronológicamente con indicadores de estado de versión y nuevas funcionalidades:

| Elemento | Descripción |
|---------|-------------|
| 👤 Mensaje de Usuario | Tus entradas (texto o voz transcrita) con timestamp |
| 🤖 Mensaje del Agente | Respuestas del agente con indicador de tiempo de procesamiento |
| 🔧 Llamada de Herramienta | Cuando el agente dispara un webhook o acción |
| ✅ Resultado de Herramienta | Respuesta del webhook o acción |
| 📝 Estado de Versión | Indicador del estado actual del agente |
| 🎤 Indicador de Voz | Muestra cuándo se usó reconocimiento de voz |

### Reproducción de Audio Mejorada

Para cada respuesta del agente, puedes:
- ▶️ **Reproducir** la respuesta de audio con controles avanzados
- ⏸️ **Pausar/Reanudar** reproducción en cualquier momento
- ⏩ **Velocidad ajustable** (0.5x, 1x, 1.5x, 2x)
- 📊 **Visualización de forma de onda** durante la reproducción
- 🔄 **Repetir mensajes** con loop automático
- 💾 **Descargar audio** para análisis offline

## Mejores Prácticas Actualizadas

<Steps>
  <Step title="Preparación Pre-Prueba">
    - Verifica el estado de sincronización del agente
    - Configura variables de contexto realistas
    - Selecciona el idioma apropiado para reconocimiento de voz
  </Step>
  <Step title="Durante las Pruebas">
    - Utiliza tanto entrada de voz como texto para cobertura completa
    - Monitorea las estadísticas de caracteres y tokens en tiempo real
    - Aprovecha los atajos de teclado para eficiencia
  </Step>
  <Step title="Análisis Post-Prueba">
    - Revisa las evaluaciones automáticas generadas
    - Analiza las métricas de rendimiento
    - Exporta conversaciones importantes para análisis posterior
  </Step>
  <Step title="Iteración y Mejora">
    - Utiliza las sugerencias de IA para optimizar el agente
    - Implementa cambios basados en métricas de fiabilidad
    - Documenta problemas encontrados y soluciones aplicadas
  </Step>
</Steps>

## Troubleshooting Avanzado

| Problema | Causa Posible | Solución |
|----------|---------------|----------|
| Reconocimiento de voz no funciona | Permisos de micrófono | Verificar permisos del navegador |
| Transcripción incorrecta | Idioma mal configurado | Cambiar idioma en configuración de voz |
| Estadísticas no se actualizan | Problema de sincronización | Refrescar el playground |
| Evaluaciones fallan | Conflictos de versión | Resolver conflictos antes de evaluar |
| Audio no se reproduce | Problema de codec | Verificar soporte de audio del navegador |
