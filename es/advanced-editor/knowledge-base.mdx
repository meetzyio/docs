## ¿Qué es la Base de Conocimiento?

La Base de Conocimiento es donde proporcionas a tu agente información que puede consultar durante las conversaciones. Esto incluye documentación de productos, FAQs, políticas de empresa, información de precios y cualquier otro contenido que tu agente necesite para responder preguntas de clientes con precisión.

<Info>
Una Base de Conocimiento bien poblada mejora significativamente la capacidad de tu agente para proporcionar respuestas precisas y útiles.
</Info>

## Añadiendo Documentos

Puedes añadir conocimiento a tu agente de cuatro formas diferentes:

### 1. Importación por URL

Importa contenido directamente desde páginas web.

<Steps>
  <Step title="Haz clic en 'Añadir URL'">
    Abre la sección de Base de Conocimiento y selecciona la opción URL.
  </Step>
  <Step title="Ingresa la URL">
    Pega la URL completa de la página web que quieres importar.
  </Step>
  <Step title="Configura nombre y modo">
    Opcionalmente añade un nombre personalizado y selecciona el modo de uso.
  </Step>
  <Step title="Confirma Importación">
    El sistema buscará y procesará el contenido automáticamente.
  </Step>
</Steps>

**Mejor para:** Páginas de productos, artículos del centro de ayuda, sitios de documentación.

**Ejemplos:**
- `https://miempresa.com/productos/plan-premium`
- `https://soporte.miempresa.com/politicas-devolucion`
- `https://docs.miempresa.com/api/authentication`

### 2. Entrada de Texto

Ingresa o pega contenido de texto manualmente.

<Steps>
  <Step title="Haz clic en 'Añadir Texto'">
    Selecciona la opción de entrada de texto.
  </Step>
  <Step title="Ingresa Contenido">
    Escribe o pega tu contenido en el área de texto.
  </Step>
  <Step title="Configura nombre y modo">
    Añade un nombre descriptivo y selecciona el modo de uso.
  </Step>
  <Step title="Guardar">
    El contenido es procesado y añadido a la base de conocimiento.
  </Step>
</Steps>

**Mejor para:** FAQs personalizadas, guías internas, scripts específicos.

**Ejemplo de contenido de texto:**
```
PRECIOS DE SERVICIOS 2024

Consulta Básica: €50
- Duración: 30 minutos
- Incluye: Evaluación inicial
- Disponible: Lunes a Viernes

Consulta Premium: €120
- Duración: 60 minutos
- Incluye: Evaluación + Plan personalizado
- Disponible: 7 días a la semana
```

### 3. Subida de Archivos

Sube documentos directamente desde tu computadora.

<Steps>
  <Step title="Haz clic en 'Subir Archivo'">
    Selecciona la opción de subida de archivo.
  </Step>
  <Step title="Selecciona Archivo">
    Elige un archivo de tu ordenador compatible.
  </Step>
  <Step title="Configura nombre y modo">
    Opcionalmente modifica el nombre y selecciona el modo de uso.
  </Step>
  <Step title="Subir">
    El documento es procesado, extraído e indexado automáticamente.
  </Step>
</Steps>

**Formatos soportados:**
- **Documentos:** PDF, DOCX, TXT, MD
- **Hojas de cálculo:** CSV
- **Formatos web:** HTML, EPUB
- **Otros formatos de texto estructurado**

<Warning>
Los archivos se convierten automáticamente a texto para su procesamiento. Asegúrate de que el contenido sea legible y esté bien estructurado.
</Warning>

### 4. Importación Múltiple por URLs

<Info>
Esta funcionalidad permite importar múltiples páginas de un sitio web de forma masiva, ideal para documentación extensa.
</Info>

Importa múltiples páginas web relacionadas en una sola operación.

<Steps>
  <Step title="Haz clic en 'Importación Múltiple'">
    Selecciona la opción de importación masiva.
  </Step>
  <Step title="Configura la importación">
    - Ingresa la URL base del sitio
    - Define patrones de URL a incluir/excluir
    - Establece límites de profundidad
  </Step>
  <Step title="Previsualiza las URLs">
    Revisa la lista de páginas que se importarán.
  </Step>
  <Step title="Ejecuta la importación">
    El sistema procesará todas las páginas automáticamente.
  </Step>
</Steps>

**Casos de uso:**
- Documentación técnica completa
- Catálogos de productos extensos
- Bases de conocimiento corporativas
- Sitios web de soporte al cliente

## Modos de Uso

Cada documento en tu Base de Conocimiento puede configurarse con uno de estos modos:

<Tabs>
  <Tab title="Búsqueda Automática">
    **🔍 Búsqueda Automática (Recomendado)**
    
    El agente busca en este documento solo cuando es relevante para la consulta del usuario. Utiliza búsqueda semántica inteligente para recuperar información contextualmente apropiada.
    
    **Ventajas:**
    - Respuestas más eficientes y focalizadas
    - Menor costo operativo
    - Mejor rendimiento para documentos grandes
    - Información recuperada solo cuando es necesaria
    
    **Ideal para:**
    - Catálogos de productos extensos
    - Documentación técnica completa
    - FAQs amplias (>50 preguntas)
    - Manuales de usuario
    - Bases de datos de conocimiento
    
    **Ejemplo:** Un catálogo con 200 productos donde el agente solo recupera información del "Plan Premium" cuando el usuario pregunta específicamente sobre ese plan.
  </Tab>
  
  <Tab title="En Prompt">
    **📋 En Prompt (Información Crítica)**
    
    El contenido siempre se incluye en el contexto del agente para cada conversación. Esta información estará inmediatamente disponible sin necesidad de búsqueda.
    
    **Ventajas:**
    - Acceso instantáneo a información crítica
    - Respuestas más consistentes
    - Ideal para datos que el agente debe conocer siempre
    
    **Limitaciones:**
    - Aumenta el costo por conversación
    - Puede ralentizar las respuestas si se abusa
    - Límite recomendado: máximo 3,000 caracteres
    
    **Ideal para:**
    - Información corporativa esencial (nombre, misión)
    - Horarios de atención
    - Datos de contacto principales
    - Políticas críticas que siempre deben mencionarse
    - Precios principales o ofertas actuales
    
    **Ejemplo:** 
    ```
    INFORMACIÓN ESENCIAL DE LA EMPRESA
    
    Nombre: TechSolutions España
    Horario: Lunes a Viernes 9:00-18:00
    Teléfono: 91 234 5678
    Email: contacto@techsolutions.es
    
    PROMOCIÓN ACTUAL: 20% descuento en todos 
    los planes hasta el 31 de diciembre.
    ```
  </Tab>
</Tabs>

<Warning>
**Importante:** Usar "En Prompt" para documentos grandes (>3,000 caracteres) puede aumentar significativamente los costos y ralentizar las respuestas. Úsalo solo para información esencial que el agente debe conocer en cada conversación.
</Warning>

## Editando Documentos

Puedes editar cualquier documento después de añadirlo a tu base de conocimiento. El proceso varía según el tipo de documento:

### Modal de Edición

Cuando haces clic en el botón **Editar** (✏️) de cualquier documento, se abre un modal de edición que te permite:

<Tabs>
  <Tab title="Documentos de Texto">
    **Edición Completa de Contenido**
    
    Para documentos de texto, puedes modificar todo el contenido directamente:
    
    - **Campo de texto amplio:** Editor con formato que permite editar todo el contenido
    - **Vista previa en tiempo real:** Ve cómo se mostrará el documento
    - **Historial de cambios:** Sistema de versiones para rastrear modificaciones
    
    **Casos de uso:**
    - Actualizar FAQs con nuevas preguntas
    - Modificar información de productos
    - Corregir errores en políticas internas
    - Añadir información estacional o promocional
    
    ```markdown
    Ejemplo de edición:
    
    Original:
    "Nuestro horario es 9:00-17:00"
    
    Editado:
    "Nuestro horario es 9:00-18:00 (ampliado desde enero 2024)
    Fines de semana: Solo citas de urgencia"
    ```
  </Tab>
  
  <Tab title="Documentos de URL">
    **Edición de URL y Recarga de Contenido**
    
    Para documentos importados desde URLs, puedes:
    
    - **Cambiar la URL:** Actualizar la dirección web fuente
    - **Recargar contenido:** Obtener la versión más reciente de la página
    - **Editar contenido:** Modificar el texto importado si es necesario
    
    **Funcionalidades especiales:**
    - **Recarga automática:** Botón para actualizar contenido sin cambiar URL
    - **Validación de URL:** Verificación de que la nueva URL es accesible
    - **Comparación de cambios:** Ve qué ha cambiado en la página web
    
    **Ejemplo de flujo:**
    ```
    URL original: https://empresa.com/precios-2023
    ↓
    URL actualizada: https://empresa.com/precios-2024
    ↓ 
    Sistema recarga contenido automáticamente
    ↓
    Base de conocimiento actualizada con nuevos precios
    ```
  </Tab>
  
  <Tab title="Documentos de Archivo">
    **Edición Limitada**
    
    Para archivos subidos, las opciones de edición son más limitadas:
    
    - **No se puede editar contenido:** El contenido extraído del archivo no es modificable
    - **Solo metadatos:** Puedes cambiar nombre y modo de uso
    - **Reemplazo necesario:** Para cambiar contenido, sube un nuevo archivo
    
    **Alternativas para actualizar:**
    1. Elimina el archivo existente
    2. Sube la versión actualizada del documento
    3. O convierte a texto y edita libremente
    
    **Conversión a texto:**
    - Copia el contenido extraído
    - Crea nuevo documento de texto
    - Elimina el archivo original
    - Ahora puedes editar libremente
  </Tab>
</Tabs>

### Campos Editables Comunes

Todos los tipos de documentos permiten editar estos campos:

<Steps>
  <Step title="Nombre del Documento">
    **Campo:** Nombre personalizado
    
    - Identificación clara en la lista
    - Facilita búsqueda y organización
    - No afecta el contenido, solo la etiqueta
    
    **Buenas prácticas:**
    ```
    ❌ Evitar: "doc1.pdf"
    ✅ Mejor: "Política de Devoluciones 2024"
    ✅ Mejor: "FAQ - Facturación y Pagos"
    ✅ Mejor: "Catálogo Productos Premium"
    ```
  </Step>
  
  <Step title="Modo de Uso">
    **Selector:** En Prompt vs Búsqueda Automática
    
    - Cambia cómo el agente accede a la información
    - Impacta costos y velocidad de respuesta
    - Se puede cambiar en cualquier momento
    
    **Ejemplo de cambio:**
    ```
    Documento: "Precios Principales"
    Cambio: Búsqueda Automática → En Prompt
    Razón: Queremos que el agente siempre mencione 
           nuestros precios principales
    ```
  </Step>
  
  <Step title="Contenido (según tipo)">
    **Variable según origen:**
    
    - **Texto:** Editor completo con formato
    - **URL:** Campo de URL + editor de contenido
    - **Archivo:** Solo metadatos (no contenido)
    
    **Limitaciones por tipo:**
    - Archivos PDF: No editables directamente
    - URLs dinámicas: Contenido puede cambiar al recargar
    - Texto: Completamente editable
  </Step>
</Steps>

### Proceso de Guardar Cambios

<Steps>
  <Step title="Validación Automática">
    El sistema verifica que todos los campos obligatorios estén completos:
    
    - **Nombre:** No puede estar vacío
    - **URL:** Debe ser válida y accesible (para documentos URL)
    - **Contenido:** Debe tener al menos 10 caracteres (para texto)
  </Step>
  
  <Step title="Vista Previa de Cambios">
    Antes de guardar, puedes revisar:
    
    - Qué campos han cambiado
    - Impacto en el modo de uso
    - Preview del contenido actualizado
  </Step>
  
  <Step title="Procesamiento">
    Una vez guardados los cambios:
    
    - **Actualización inmediata:** Los metadatos se actualizan al instante
    - **Procesamiento de contenido:** El contenido nuevo se indexa
    - **Disponibilidad:** Cambios activos en 1-2 minutos
  </Step>
  
  <Step title="Confirmación">
    El sistema confirma que los cambios se han guardado exitosamente:
    
    - Mensaje de éxito en la interfaz
    - El documento actualizado aparece en la lista
    - Se cierra automáticamente el modal de edición
  </Step>
</Steps>

### Casos de Uso Comunes de Edición

**1. Actualización Estacional**
```markdown
Documento: "Horarios de Atención"

Edición (Diciembre):
Horario regular: Lunes-Viernes 9:00-18:00
Horario navideño (24-31 Dic): 9:00-14:00
Cerrado: 25 Dic, 1 Enero

Modo: En Prompt (información crítica)
```

**2. Corrección de Errores**
```markdown
Documento: "Precios Servicios"

Error detectado: "Consulta Premium: €50"
Corrección: "Consulta Premium: €120"

Edición inmediata vía modal de texto
```

**3. Expansión de Información**
```markdown
Documento: "FAQ Facturación"

Adición de nueva pregunta:
P: ¿Aceptan pagos fraccionados?
R: Sí, ofrecemos planes de pago en 3 cuotas 
   sin intereses para servicios superiores a €200.

Modo: Búsqueda Automática
```

**4. Actualización de URLs**
```markdown
URL obsoleta: https://empresa.com/productos-2023
Nueva URL: https://empresa.com/productos-2024

Acción:
1. Editar documento
2. Cambiar URL
3. Sistema recarga contenido automáticamente
4. Base de conocimiento actualizada
```

### Seguimiento de Cambios

<Info>
El sistema mantiene un registro de cambios para ayudarte a rastrear modificaciones y revertir si es necesario.
</Info>

**Información rastreada:**
- Fecha y hora de cada modificación
- Qué campos fueron cambiados
- Usuario que realizó los cambios
- Versión anterior del contenido

**Acceso al historial:**
- Visible en las propiedades del documento
- Útil para auditorías y control de versiones
- Permite identificar cuándo se introdujeron cambios específicos

## Gestionando Documentos

### Vista de Lista de Documentos

Todos los documentos añadidos aparecen en una lista organizada que muestra:

| Elemento | Descripción | Información Mostrada |
|----------|-------------|---------------------|
| **Icono de Tipo** | Identifica el origen del documento | 🔗 URL, 📄 Texto, 📎 Archivo |
| **Nombre** | Título del documento o URL | Nombre personalizado o generado automáticamente |
| **Modo de Uso** | Cómo utiliza el agente esta información | 🔍 Búsqueda Automática / 📋 En Prompt |
| **Vista Previa** | Primeras líneas del contenido | Texto truncado para identificación rápida |
| **Estado** | Información de procesamiento | ✅ Procesado / ⚠️ Procesando / ❌ Error |
| **Acciones** | Botones de gestión | Editar (✏️), Eliminar (🗑️), Recargar (🔄) |

### Funciones Avanzadas de Gestión

#### Recargar Contenido de URL
Para URLs, puedes recargar el contenido para obtener la versión más actualizada:
- Haz clic en el icono de recarga (🔄)
- El sistema volverá a descargar y procesar la página
- Útil para contenido que cambia frecuentemente

#### Búsqueda y Filtrado
- **Búsqueda por nombre:** Encuentra documentos rápidamente
- **Filtro por tipo:** Muestra solo URLs, archivos o texto
- **Filtro por modo:** Separa documentos en prompt vs búsqueda automática

#### Eliminación Segura
<Warning>
Los documentos eliminados no pueden recuperarse. El sistema te pedirá confirmación antes de eliminar cualquier documento.
</Warning>

1. Haz clic en el icono de eliminar (🗑️) junto al documento
2. Confirma la eliminación en el diálogo de confirmación
3. El documento se elimina permanentemente de la base de conocimiento

## Mejores Prácticas

### Estructuración de Contenido

<Info>
Organiza tu información usando encabezados claros, listas y formato estructurado para que el agente pueda extraer datos más eficientemente.
</Info>

#### Para Documentos de Texto
```markdown
# PREGUNTAS FRECUENTES - PLANES DE SERVICIO

## Plan Básico
**Precio:** €29/mes
**Características:**
- 5 proyectos incluidos
- Soporte por email
- 10GB de almacenamiento

## Plan Professional  
**Precio:** €79/mes
**Características:**
- Proyectos ilimitados
- Soporte prioritario 24/7
- 100GB de almacenamiento
- Acceso API completo

## Políticas de Cancelación
- Cancelación gratuita en cualquier momento
- Reembolso prorrateado en primeros 30 días
- Los datos se conservan 90 días tras cancelación
```

#### Para URLs
- Asegúrate de que las páginas tengan estructura HTML semántica
- Evita páginas con mucho contenido dinámico o JavaScript
- Prefiere páginas con contenido textual claro sobre imágenes

#### Para Archivos
- Usa documentos bien formateados (títulos, subtítulos)
- Evita archivos escaneados de baja calidad
- PDFs nativos funcionan mejor que PDFs escaneados

### Estrategias de Organización

1. **Principio de Especificidad**
   - Información general → "En Prompt"
   - Información específica → "Búsqueda Automática"

2. **Evita Duplicación**
   - No repitas la misma información en múltiples documentos
   - Usa un documento maestro para información que aparece en varios contextos

3. **Mantenimiento Regular**
   - Revisa y actualiza documentos mensualmente
   - Elimina información obsoleta
   - Recarga URLs que cambian frecuentemente

4. **Estructura Jerárquica**
   ```
   📚 Base de Conocimiento
   ├── 📄 Información Corporativa (En Prompt)
   │   └── Datos esenciales, horarios, contacto
   ├── 🔍 Productos y Servicios (Búsqueda Automática)
   │   ├── Catálogo completo
   │   ├── Precios detallados
   │   └── Especificaciones técnicas
   ├── 🔍 Soporte al Cliente (Búsqueda Automática)
   │   ├── FAQs extensas
   │   ├── Guías de solución de problemas
   │   └── Políticas de devolución
   └── 📄 Promociones Actuales (En Prompt)
       └── Ofertas vigentes y códigos de descuento
   ```

### Optimización para IA

1. **Usa Lenguaje Natural Conversacional**
   ```
   ❌ Incorrecto: "PRC-001: €50 - 30min - M-F"
   ✅ Correcto: "La Consulta Premium cuesta €50, dura 30 minutos y está disponible de lunes a viernes."
   ```

2. **Incluye Contexto y Ejemplos**
   ```markdown
   ## Política de Reembolsos
   
   Ofrecemos reembolsos completos dentro de los primeros 30 días de compra.
   
   **Ejemplo:** Si compras nuestro Plan Professional el 1 de enero, 
   puedes solicitar un reembolso completo hasta el 31 de enero.
   
   **Para solicitar reembolso:**
   1. Envía email a reembolsos@miempresa.com
   2. Incluye número de orden
   3. Recibirás confirmación en 24 horas
   ```

3. **Anticipa Variaciones en Preguntas**
   ```markdown
   ## Horarios de Atención
   
   Nuestro horario de atención es de lunes a viernes de 9:00 a 18:00 (hora de España).
   
   **También conocido como:**
   - Horario de oficina
   - Cuando están abiertos
   - Horas de servicio al cliente
   - Cuándo puedo llamar
   
   **Nota:** Los fines de semana y festivos españoles permanecemos cerrados.
   ```

## Monitoreo y Optimización

### Análisis de Uso

El sistema rastrea automáticamente:
- **Documentos más consultados:** Identifica qué información es más valiosa
- **Consultas sin respuesta:** Detecta gaps en tu base de conocimiento
- **Tiempo de búsqueda:** Mide la eficiencia de recuperación de información

### Métricas de Rendimiento

<Info>
Puedes acceder a métricas detalladas en la sección de Analytics del agente para ver cómo se utiliza tu base de conocimiento.
</Info>

- **Tasa de cobertura:** % de preguntas que el agente puede responder con tu contenido
- **Precisión de búsqueda:** Qué tan relevante es la información recuperada
- **Velocidad de respuesta:** Impacto en el tiempo de respuesta del agente

### Señales para Optimización

**Cuando añadir más contenido:**
- El agente responde "No tengo información sobre eso" frecuentemente
- Las consultas de clientes revelan gaps de conocimiento
- Nuevos productos/servicios que no están documentados

**Cuando simplificar:**
- Respuestas del agente son demasiado largas o confusas
- Documentos "En Prompt" superan los 3,000 caracteres
- Tiempo de respuesta es lento por exceso de contexto

## Solución de Problemas Comunes

### Error: "No se pudo procesar el documento"

**Causas comunes:**
- Archivo corrupto o protegido con contraseña
- URL inaccesible o que requiere autenticación
- Formato de archivo no soportado
- Contenido demasiado extenso (>50MB)

**Soluciones:**
1. Verifica que el archivo no esté dañado
2. Para PDFs protegidos, elimina la contraseña primero
3. Asegúrate de que las URLs sean públicamente accesibles
4. Divide documentos muy grandes en secciones más pequeñas

### Problema: "El agente no encuentra información que sé que está en la base de conocimiento"

**Diagnóstico:**
1. Verifica que el documento esté en modo "Búsqueda Automática"
2. Comprueba que el contenido sea textualmente claro
3. Asegúrate de que la pregunta use términos similares al contenido

**Optimizaciones:**
- Reformula el contenido usando sinónimos comunes
- Añade secciones de "Preguntas relacionadas"
- Usa formato estructurado con encabezados claros

### Problema: "Las respuestas del agente son inconsistentes"

**Posibles causas:**
- Información contradictoria entre documentos
- Demasiado contenido en modo "En Prompt"
- Información obsoleta mezclada con actual

**Soluciones:**
1. Audita todos los documentos buscando contradicciones
2. Mueve contenido extenso a modo "Búsqueda Automática"
3. Elimina o actualiza información desactualizada
