## Resumen

La Configuración Básica establece los **Componentes Fundacionales** de tu agente conversacional de IA, incluyendo la configuración de infraestructura telefónica con cuentas Twilio, gestión de números de teléfono y caller IDs, así como la personalización con agentes de voz especializados por industria y capacidades conversacionales avanzadas.

## Componentes Base de Meetzy

<Info>
Los Componentes Base están habilitados por defecto en nuevos agentes para asegurar conversaciones de alta calidad desde el primer momento.
</Info>

Los Componentes Base de Meetzy proporcionan la fundación para agentes conversacionales profesionales y consistentes. Esta funcionalidad permite un control granular sobre qué elementos base del sistema Meetzy se incluyen en el prompt del asistente.

### 🛡️ Guardrails Fundacionales
**Protección y Límites Profesionales:**
- Restricciones éticas y de comportamiento
- Mantenimiento del contexto profesional
- Prevención de temas inapropiados
- Límites de conversación seguros
- Protocolos de escalación automática

### 🎭 Aspectos Conversacionales
**Habilidades de Comunicación Natural:**
- Manejo inteligente de interrupciones
- Gestión profesional de silencios
- Transiciones conversacionales fluidas
- Técnicas de clarificación activa
- Adaptación al ritmo del interlocutor

### 💼 Background Laboral de Agentes
**Experiencia Profesional Integrada:**
- Conocimiento de protocolos telefónicos empresariales
- Mejores prácticas del sector de atención
- Experiencia en comunicación corporativa
- Estándares de servicio al cliente
- Competencias profesionales transversales

<Warning>
**Importante**: Deshabilitar los Componentes Base puede resultar en conversaciones menos naturales o comportamientos impredecibles. Solo desactívalos si tienes un sistema de prompts personalizado completamente desarrollado y probado.
</Warning>

<Tabs>

    **Consistencia vocal:**
    - **Rango**: 0.00 - 1.00 (por defecto: 0.15)
    - **Baja** (0.00-0.30): Voz más expresiva y variable
    - **Media** (0.31-0.70): Balance entre expresividad y consistencia
    - **Alta** (0.71-1.00): Voz muy consistente y predecible
    
    <Warning>

## Gestión de Infraestructura Telefónica

### Cuentas Twilio

<Info>
**Solo usuarios administradores**: La selección de cuentas Twilio está disponible únicamente para usuarios con permisos administrativos.
</Info>

Los administradores pueden seleccionar diferentes cuentas Twilio para organizar y distribuir recursos telefónicos:

<Steps>
  <Step title="Seleccionar cuenta Twilio">
    Elige la cuenta Twilio que proporcionará los recursos para este agente
  </Step>
  <Step title="Cargar números disponibles">
    Los números de teléfono se actualizarán automáticamente según la cuenta seleccionada
  </Step>
  <Step title="Verificar compatibilidad">
    Asegúrate de que la cuenta tenga números disponibles para asignación
  </Step>
</Steps>

### Gestión de Números de Teléfono

<Tabs>
  <Tab title="Agentes Nuevos">
    <Steps>
      <Step title="Seleccionar número disponible">
        Elige de la lista de números no asignados del menú desplegable
      </Step>
      <Step title="Verificar disponibilidad">
        Solo se muestran números que no están en uso por otros agentes
      </Step>
      <Step title="Solicitar números adicionales">
        Usa "Request more numbers" si necesitas más opciones disponibles
      </Step>
    </Steps>
  </Tab>
  <Tab title="Agentes Existentes">
    El número asignado se muestra como solo lectura y no puede modificarse después de la creación del agente.

    Cambiar números en agentes existentes requiere crear un nuevo agente o contactar soporte técnico.
    </Warning>
  </Tab>
</Tabs>

### Gestión de Caller IDs (Números de Salida)

Los Caller IDs permiten configurar números específicos para llamadas salientes, mejorando la confianza del destinatario:

#### Añadir Nuevo Caller ID

<Steps>
  <Step title="Abrir modal de adición">
    Haz clic en "Add Caller ID" para iniciar el proceso
  </Step>
  <Step title="Introducir información">
    - **Nombre**: Identificación descriptiva del número
    - **Teléfono**: Número en formato E.164 (+1234567890)
  </Step>
  <Step title="Iniciar validación">
    Twilio enviará una llamada automática con código de verificación
  </Step>
  <Step title="Introducir código">
    Responde la llamada e introduce el código de 6 dígitos recibido
  </Step>
  <Step title="Completar verificación">
    El número queda disponible para llamadas salientes una vez verificado
  </Step>
</Steps>

#### Estados de Caller ID

<Tabs>
  <Tab title="Verificado">
    - **Estado**: ✅ Verified
    - **Disponibilidad**: Listo para usar en llamadas salientes
    - **Acciones**: Eliminar (con confirmación)
  </Tab>
  <Tab title="Pendiente">
    - **Estado**: ⏳ Pending
    - **Disponibilidad**: En proceso de verificación
    - **Acciones**: Reenviar código, Eliminar
  </Tab>
  <Tab title="Fallido">
    - **Estado**: ❌ Failed
    - **Disponibilidad**: Requiere nueva verificación
    - **Acciones**: Reintentar verificación, Eliminar
  </Tab>
</Tabs>

<Warning>
Los Caller IDs no verificados no pueden utilizarse para llamadas salientes. Completa siempre el proceso de verificación.
</Warning>

## Troubleshooting Avanzado

<Tabs>
  <Tab title="Problemas con Caller IDs">
    **Síntomas Comunes:**
    - Verificación fallida repetidamente
    - Número no recibe códigos de verificación
    - Estados de validación inconsistentes
    
    **Soluciones Detalladas:**
    - Verificar formato exacto E.164 (+país + número)
    - Confirmar que el número puede recibir llamadas
    - Revisar configuración de operador telefónico
    - Intentar desde diferentes ubicaciones/dispositivos
    - Contactar soporte para números problemáticos
  </Tab>
  
  <Tab title="Números de Teléfono">
    **Síntomas Comunes:**
    - Lista de números completamente vacía
    - Números deseados no aparecen disponibles
    - Errores de carga de números
    
    **Soluciones Sistemáticas:**
    - Verificar selección de cuenta Twilio correcta
    - Solicitar más números usando "Request more numbers"
    - Contactar administrador para liberación de números asignados
    - Verificar permisos de cuenta Twilio
    - Revisar límites de números por cuenta
  </Tab>
  
  <Tab title="Rendimiento de Agentes">
    **Síntomas Comunes:**
    - Respuestas lentas o inconsistentes
    - Calidad conversacional baja
    - Comportamientos inesperados del agente
    
    **Optimización Avanzada:**
    - Cambiar a modelo más rápido para mejorar velocidad
    - Usar modelo más potente para conversaciones complejas
    - Ajustar prompts de sistema específicos
    - Revisar configuración de agente especializado
    - Analizar logs de conversaciones para patrones
  </Tab>
  
  <Tab title="Problemas de Integración">
    **Síntomas Comunes:**
    - Fallas en conexión con sistemas externos
    - Sincronización de datos incorrecta
    - Errores de API recurrentes
    
    **Diagnóstico y Solución:**
    - Verificar credenciales y tokens de API
    - Revisar configuración de webhooks
    - Comprobar conectividad de red y firewalls
    - Validar formatos de datos enviados/recibidos
    - Contactar soporte técnico con logs específicos
  </Tab>
</Tabs>

<Warning>
**Crítico**: Cambios en configuraciones fundamentales (modelo de IA, número de teléfono, agente de voz especializado) pueden afectar significativamente el comportamiento y rendimiento del agente. Siempre realiza pruebas exhaustivas en el Playground después de cualquier modificación importante antes de implementar en producción.
</Warning>

## Siguientes Pasos

Después de configurar los aspectos fundamentales de tu agente especializado, continúa personalizando su comportamiento y capacidades:

- **Saludos Personalizados**: Configura mensajes de bienvenida específicos para llamadas entrantes y salientes adaptados a tu agente especializado
- **Prompt del Sistema**: Define la personalidad específica, comportamiento y objetivos del agente especializado
- **Acciones del Agente**: Configura transferencias inteligentes de llamadas, envío de emails y SMS automatizados
- **Playground de Pruebas**: Prueba y valida exhaustivamente tu agente especializado antes del despliegue en producción
