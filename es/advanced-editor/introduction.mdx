## ¿Qué es el Editor Avanzado?

El **Editor Avanzado** es la herramienta integral de configuración de agentes de Meetzy que te da control total sobre cada aspecto del comportamiento de tu agente de IA. Proporciona una interfaz de nivel profesional para crear, probar y desplegar agentes conversacionales de IA con capacidades avanzadas de colaboración y testing.

<Info>
El Editor Avanzado incluye ahora un **flujo de creación prompt-first** que permite generar agentes automáticamente desde una descripción, junto con un **selector de agentes integrado** para navegación fluida entre múltiples agentes sin perder contexto de trabajo.
</Info>

## Creación Prompt-First de Agentes

Una de las innovaciones más recientes del Editor Avanzado es la capacidad de generar agentes automáticamente mediante descripción en lenguaje natural.

### Cómo Funciona

Cuando accedes al Editor Avanzado para crear un nuevo agente, se presenta una experiencia de **generación guiada por IA**:

<Steps>
  <Step title="Interfaz de Creación Intuitiva">
    Una pantalla inmersiva con animación de ondas WebGL donde describes qué debe hacer tu agente
  </Step>
  <Step title="Procesamiento por IA">
    Meetzy analiza tu descripción y genera automáticamente toda la configuración del agente
  </Step>
  <Step title="Resultado Completo">
    Obtén un agente funcional con prompt del sistema, configuraciones, herramientas y flujos de conversación
  </Step>
</Steps>

### Ejemplos de Prompts Efectivos

| Tipo de Agente | Prompt de Ejemplo |
|----------------|-------------------|
| **Soporte Técnico** | "Un agente que resuelve problemas técnicos de software, escalando casos complejos al equipo humano cuando es necesario" |
| **Ventas** | "Agente de ventas para una tienda de moda online que ayuda a clientes a encontrar productos, procesa pedidos y maneja devoluciones" |
| **Atención Médica** | "Asistente para clínica dental que agenda citas, responde preguntas sobre tratamientos y transfiere emergencias al doctor" |
| **Educación** | "Tutor virtual que ayuda a estudiantes con matemáticas de bachillerato, adapta explicaciones al nivel del estudiante" |

<Warning>
**Generación Inteligente**: Cuanto más específico y detallado sea tu prompt inicial, más preciso será el agente generado. Incluye información sobre el contexto, funcionalidades específicas y casos de uso principales.
</Warning>

## Nuevo Selector de Agentes

Una de las características más útiles del Editor Avanzado es el **selector de agentes integrado** en la barra superior, que permite una navegación fluida entre múltiples agentes:

### Funcionalidades del Selector

| Característica | Descripción |
|----------------|-------------|
| **Navegación rápida** | Cambia entre agentes sin perder el contexto de edición |
| **Búsqueda inteligente** | Filtrado en tiempo real por nombre de agente |
| **Vista previa de información** | Muestra nombre completo e ID del agente |
| **Estado visual** | Indicador de selección del agente activo |
| **Carga asíncrona** | Lista de agentes actualizada en tiempo real |
| **Filtro por empresa** | Para administradores: filtrar agentes por organización |

### Interfaz Visual del Selector

El selector de agentes aparece en la esquina superior izquierda cuando estás editando un agente existente:

- **Botón principal**: Muestra el nombre del agente actual con un icono desplegable
- **Búsqueda integrada**: Campo de filtrado con icono de lupa para encontrar agentes rápidamente
- **Lista de agentes**: Tarjetas con avatar, nombre completo e ID abreviado
- **Indicador de selección**: Marca visual verde para el agente actualmente seleccionado
- **Filtro de empresa**: (Solo administradores) Dropdown para filtrar por organización

<Warning>
**Navegación Inteligente**: El sistema detecta automáticamente cambios no guardados y muestra un modal de confirmación antes de cambiar entre agentes, evitando la pérdida accidental de trabajo.
</Warning>

## Sistema de Control de Versiones Git Integrado

El Editor Avanzado incluye un sistema robusto de control de versiones Git que permite colaboración profesional en tiempo real entre equipos:

<Info>
**Colaboración en Tiempo Real**: Múltiples usuarios pueden trabajar en el mismo agente simultáneamente con detección automática de conflictos, sincronización cada 30 segundos, e indicadores de presencia visual.
</Info>

### Estados de Versión en Tiempo Real

El indicador de estado Git muestra el estado actual del agente con códigos de color intuitivos y información detallada:

| Estado | Descripción | Indicador Visual | Color |
|--------|-------------|-----------------|--------|
| **Clean** | Sin cambios pendientes, sincronizado con remoto | ✅ Clean | 🟢 Verde |
| **Modified** | Cambios locales sin confirmar | ⚠️ Modified (3) | 🟡 Amarillo |
| **Conflict** | Conflictos con cambios remotos detectados | ❌ Conflicts (2) | 🔴 Rojo |
| **Syncing** | Sincronizando con el servidor Git | 🔄 Syncing... | 🔵 Azul |
| **Locked** | Bloqueado por otros usuarios activos | 🔒 Locked by 2 user(s) | 🟠 Naranja |

### Flujo de Trabajo Git Profesional

El sistema implementa un flujo de trabajo Git completo:

#### Operaciones Git Básicas
- **Check**: Verificar actualizaciones remotas sin aplicar cambios
- **Pull**: Sincronizar con cambios remotos y resolver conflictos automáticamente
- **Commit**: Confirmar cambios locales con mensajes descriptivos obligatorios
- **Push**: Desplegar a producción tras validación automática de tests

#### Gestión Inteligente de Conflictos
- **Detección automática**: Identificación de conflictos a nivel de campo
- **Resolución visual**: Interfaz paso a paso para resolver cada conflicto
- **Preview de merge**: Vista previa del resultado antes de confirmar
- **Rollback seguro**: Capacidad de deshacer cambios problemáticos

</Warning>

### Funcionalidades Avanzadas de Testing

#### Ejecución en Lote Inteligente
- **Repeticiones múltiples**: Ejecutar el mismo test hasta 10 veces para validar consistencia
- **Análisis estadístico**: Métricas detalladas de éxito/fallo por lote con tendencias
- **Paralelización**: Ejecución simultánea para reducir tiempo de testing
- **Distribución de carga**: Balanceamiento automático de recursos de testing

#### Variables de Contexto Dinámicas
- **Parámetros dinámicos**: Inyección automática de variables como fecha_y_hora_actual
- **Personalización por test**: Variables específicas por escenario de testing
- **Gestión visual**: Panel dedicado para configurar y validar variables
- **Templating avanzado**: Sistema de plantillas para reutilización de configuraciones

#### Historial y Análisis de Performance
- **Pestaña History**: Registro completo de ejecuciones con métricas detalladas
- **Métricas de rendimiento**: Tendencias de éxito/fallo a lo largo del tiempo
- **Comparación de versiones**: Análisis del impacto de cambios en performance
- **Alertas automáticas**: Notificaciones de degradación de calidad

#### Integración con AI Copilot
- **Mejoras automáticas**: Copilot sugiere optimizaciones basadas en tests fallidos
- **Análisis de fallos**: Identificación automática de patrones de error y causas raíz
- **Optimización de prompts**: Sugerencias específicas para mejorar comportamiento

### Versionado Semántico Automático

El sistema utiliza versionado semántico con detección automática de tipo de cambio:

| Tipo de Cambio | Incremento | Ejemplo | Detección Automática |
|----------------|-----------|---------|----------------------|
| **Patches** | v1.0.0 → v1.0.1 | Corrección de typos, ajustes menores | Cambios en strings sin impacto funcional |
| **Minor** | v1.0.1 → v1.1.0 | Nuevas funcionalidades compatibles | Nuevas acciones, herramientas, o configuraciones |
| **Major** | v1.1.0 → v2.0.0 | Cambios incompatibles | Modificaciones en estructura de salida o API |

4. **Production deployment gates**: Deploy condicionado al éxito de todas las validaciones
5. **Post-deploy monitoring**: Monitoreo automático de métricas en producción

#### Métricas y Monitoreo Avanzado
- **Test coverage analytics**: Análisis de cobertura por funcionalidad y flujo
