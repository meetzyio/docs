# Historial de Llamadas

La sección de Historial de Llamadas proporciona una vista completa y detallada de todas las llamadas realizadas por tus agentes de IA. Con herramientas avanzadas de filtrado, búsqueda y análisis, puedes monitorear el rendimiento e identificar insights valiosos para optimizar tus operaciones telefónicas.

## Acceder al Historial

Navega a **Calls** en el menú principal para acceder a tu historial completo de llamadas con la nueva interfaz mejorada.

## Vista General de la Interfaz

La interfaz renovada del historial de llamadas incluye:

| Sección | Descripción |
|---------|-------------|
| 📊 **Cabecera Expandida** | Título, descripción detallada y estado de selección con acciones |
| 🔍 **Barra de Filtros Avanzada** | Sistema de filtrado modular con presets y configuración personalizable |
| 📋 **Tabla de Llamadas** | Lista interactiva con información clave y controles de visibilidad |
| 📄 **Modal de Detalles** | Vista detallada de llamadas individuales con transcripciones |
| ⚙️ **Panel de Configuración** | Control de visibilidad de columnas y presets de filtros |

## Sistema de Filtrado Avanzado

### Filtro de Rango de Fechas Mejorado

El nuevo selector de fechas ofrece mayor flexibilidad y precisión:

<Steps>
  <Step title="Presets Rápidos">
    Accede a rangos predefinidos optimizados:
    - **Today** - Llamadas del día actual
    - **Yesterday** - Llamadas del día anterior  
    - **Last 7 Days** - Llamadas de la última semana
    - **Last 30 Days** - Llamadas del último mes
    - **This Month** - Mes actual completo
    - **Last Month** - Mes anterior completo
  </Step>
  
  <Step title="Rango Personalizado">
    Define períodos específicos con precisión horaria:
    - Selector de fecha y hora inicial
    - Selector de fecha y hora final
    - Validación automática de rangos válidos
    - Aplicación inmediata con un clic
  </Step>
</Steps>

<Info>
El selector de fechas se desactiva automáticamente cuando usas la búsqueda por Call ID para evitar conflictos de filtros.
</Info>

### Filtro de Agentes con Multi-selección

Sistema mejorado de selección de agentes:

| Funcionalidad | Descripción |
|---------------|-------------|
| **Selección Múltiple** | Checkbox para cada agente disponible |
| **Indicador de Estado** | Muestra "X agents selected" cuando hay filtros activos |
| **Filtro de Búsqueda** | Campo de texto para encontrar agentes rápidamente |
| **Seleccionar Todo** | Opción para incluir todos los agentes de una vez |

### Filtros de Estado y Resultado

Filtra con precisión por diferentes criterios:

#### Estados de Llamada
| Estado | Descripción | Indicador Visual |
|--------|-------------|------------------|
| 🟢 **Completed** | Llamada finalizada correctamente | Badge verde |
| 🟡 **In Progress** | Llamada actualmente activa | Badge amarillo |
| 🔴 **Failed** | Llamada con error o fallo técnico | Badge rojo |
| ⚪ **No Answer** | El destinatario no contestó | Badge gris |
| 🔵 **Transferred** | Llamada transferida a humano | Badge azul |

#### Dirección de Llamadas
- **Inbound** - Llamadas recibidas por tus agentes
- **Outbound** - Llamadas iniciadas por tus agentes  
- **All** - Ambas direcciones sin distinción

#### Resultado de Éxito
- **Successful** - Llamadas que cumplieron los criterios de éxito
- **Unsuccessful** - Llamadas que no alcanzaron los objetivos
- **All** - Todas las llamadas independientemente del resultado

### Búsqueda por Call ID con Modo Exclusivo

<Warning>
Cuando introduces un Call ID específico, todos los demás filtros se desactivan temporalmente para mostrar únicamente la llamada exacta correspondiente a ese identificador.
</Warning>

Características de la búsqueda por ID:

1. **Entrada de ID Específico** - Campo de texto para el identificador único
2. **Búsqueda Exacta** - Coincidencia precisa sin aproximaciones
3. **Prioridad Absoluta** - Override de todos los demás filtros activos
4. **Restauración Automática** - Los filtros se reactivan al limpiar el campo

## Configuración de Visibilidad de Columnas

### Panel de Personalización

Accede al panel de configuración mediante el botón de configuración (⚙️) para:

<Tabs>
  <Tab title="Visibilidad de Columnas">

  </Tab>

    - **Preset Básico**: Columnas esenciales para vista rápida
    - **Preset Completo**: Todas las columnas disponibles
    - **Preset Analítico**: Métricas y datos de rendimiento
    - **Presets Personalizados**: Configuraciones específicas del usuario
  </Tab>
  
  <Tab title="Densidad de Datos">
    Ajusta la cantidad de información por fila:
    - **Compacta**: Máximo número de filas visibles
    - **Estándar**: Balance entre información y legibilidad  
    - **Espaciada**: Mayor claridad con más espacio vertical
  </Tab>
</Tabs>

## Presets de Filtros Guardados

### Administración de Presets

<Warning>
Los presets guardados incluyen toda la configuración de filtros y visibilidad de columnas, no solo los filtros.
</Warning>

Funcionalidades disponibles:

1. **Crear Preset**
   - Configura filtros y columnas según necesites
   - Asigna un nombre descriptivo
   - Guarda para uso futuro

2. **Aplicar Preset**
   - Selección desde dropdown dedicado
   - Aplicación instantánea de toda la configuración
   - Indicador visual del preset activo

3. **Administrar Presets**
   - Editar presets existentes
   - Eliminar presets obsoletos
   - Compartir presets con el equipo

### Presets Predefinidos del Sistema

| Preset | Configuración | Uso Recomendado |
|--------|---------------|-----------------|
| **Vista Rápida** | Columnas básicas, últimos 7 días | Revisión diaria |
| **Análisis Completo** | Todas las columnas, últimos 30 días | Reportes mensuales |
| **Solo Exitosas** | Filtro por éxito, métricas clave | Identificar mejores prácticas |
| **Problemas** | Solo fallos, información técnica | Troubleshooting |

## Integración con Tendencias de Evaluaciones

Acceso directo a análisis de rendimiento:

<Steps>
  <Step title="Acceder desde Historial">
    Botón **"View Evaluation Trends"** disponible para cada agente específico
  </Step>
  
  <Step title="Análisis Contextual">
    Las tendencias se abren con el mismo período de tiempo activo en el historial
  </Step>
  
  <Step title="Navegación Fluida">
    Regresa al historial manteniendo todos los filtros aplicados
  </Step>
</Steps>

### Métricas de Evaluación Integradas

Las tendencias proporcionan insights detallados sobre:

- **Tasa de Éxito Global** - Porcentaje general de evaluaciones exitosas
- **Evolución Temporal** - Gráficos de rendimiento a lo largo del tiempo
- **Comparativa por Agente** - Benchmarking entre diferentes agentes
- **Patrones Estacionales** - Identificación de tendencias por días/horas
- **Alertas de Rendimiento** - Notificaciones de cambios significativos

<Info>
Las evaluaciones se actualizan en tiempo real y se sincronizan automáticamente con los datos del historial de llamadas.
</Info>

## Búsquedas Inteligentes y Consejos

### Estrategias de Búsqueda Eficientes

<CardGroup cols={2}>
  <Card title="Usa Presets Inteligentes" icon="bookmark">
    Comienza con presets predefinidos y personaliza según necesites. Esto ahorra tiempo y garantiza consistencia.
  </Card>
  
  <Card title="Combina Filtros Gradualmente" icon="filter">
    Aplica filtros de forma progresiva: primero fecha, luego agente, después estado. Esto te ayuda a refinar resultados paso a paso.
  </Card>
  
  <Card title="Aprovecha la Búsqueda por ID" icon="hashtag">
    Para debugging o soporte, usa directamente el Call ID exacto. Es la forma más rápida de encontrar llamadas específicas.
  </Card>
  
  <Card title="Guarda Configuraciones Útiles" icon="save">
    Crea presets para búsquedas recurrentes como reportes semanales, análisis de problemas, o revisiones de calidad.
  </Card>
</CardGroup>

### Atajos de Teclado

| Atajo | Acción |
|-------|--------|
| `Ctrl + F` | Enfocar búsqueda por Call ID |
| `Ctrl + A` | Seleccionar todas las llamadas visibles |
| `Ctrl + D` | Deseleccionar todas |
| `Ctrl + E` | Exportar selección actual |
| `Ctrl + R` | Resetear todos los filtros |

## Indicadores de Rendimiento en Tiempo Real

### Dashboard Integrado

La nueva interfaz incluye métricas en vivo:

- **Llamadas del Día**: Contador en tiempo real
- **Tasa de Éxito**: Porcentaje actualizado automáticamente  
- **Duración Promedio**: Cálculo dinámico basado en filtros

### Alertas y Notificaciones

<Warning>
El sistema monitorea automáticamente patrones inusuales y genera alertas cuando detecta:
- Caídas significativas en la tasa de éxito
- Aumentos inesperados en la duración de llamadas
- Incrementos en llamadas fallidas
- Patrones de uso anómalos
</Warning>
