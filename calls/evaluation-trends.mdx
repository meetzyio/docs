The **Evaluation Trends** page provides comprehensive insights into how your agents' performance evolves over time. Monitor evaluation patterns, celebrate improvements, and identify areas that need attention through interactive charts and detailed analytics.

## Getting Started

<Steps>
<Step title="Select an Agent">
Use the agent search field to find and select the agent you want to analyze. The search supports partial name matching for easy navigation.
</Step>

<Step title="Configure Time Range">
Choose from predefined time ranges (7 days, 30 days, 90 days, 6 months, 1 year) or set a custom date range to focus on specific periods.
</Step>

<Step title="Set Grouping Period">
Select how to group your data - by day, week, or month - depending on your analysis needs and the selected time range.
</Step>

<Step title="View Evaluations">
Click "View evaluations" to load the trend analysis and detailed metrics for your selected agent.
</Step>
</Steps>

## Key Metrics Overview

Once you've selected an agent and loaded the evaluation data, you'll see three main summary cards:

### Total Evaluations
The total number of evaluations conducted for the selected agent within your chosen time range. This gives you a quick overview of evaluation activity volume.

### Overall Success Rate
The average success rate across all evaluations, displayed as a percentage. This metric helps you understand the general performance level of your agent.

### Calls Evaluated  
The number of individual calls that have been evaluated, providing context for the total evaluation count.

<Info>
These summary metrics update automatically when you change your time range or grouping period, giving you real-time insights into different aspects of your agent's performance.
</Info>

## Detailed Evaluation Tables

Below the trend charts, you'll find comprehensive tables that include:

- **Individual evaluation records** with timestamps
- **Success/failure status** for each evaluation
- **Mini sparkline charts** showing recent performance trends
- **Sortable columns** for custom data organization

<Warning>
The evaluation trends are calculated based on available data within your selected time range. Ensure you have sufficient evaluation history for meaningful trend analysis.
</Warning>

## Best Practices

### Analyzing Trends
- **Compare different time periods** to understand seasonal patterns
- **Look for correlation** between evaluation frequency and success rates
- **Monitor trends after making agent improvements** to measure impact

### Optimal Time Ranges
- Use **shorter periods (7-30 days)** for recent performance monitoring
- Use **longer periods (3-12 months)** for strategic planning and pattern recognition
- **Custom ranges** work well for analyzing specific campaigns or time periods

### Grouping Recommendations
- **Daily grouping** for detailed short-term analysis
- **Weekly grouping** for medium-term trend identification  
- **Monthly grouping** for long-term strategic insights

<Info>
Trend data updates in real-time as new evaluations are completed, so you can monitor improvements immediately after implementing changes to your agents.
</Info>
