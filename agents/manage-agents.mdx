### Quick Agent Selector: In-Depth Guide

The agent name button in the top-left corner of the Advanced Editor's header toolbar is more than a label — it's a fully-featured quick switcher that lets you jump between agents without ever leaving the editor.

<Steps>
  <Step title="Open the Selector">
    Click on the current agent's name and colored initial avatar in the header toolbar. A dropdown panel opens directly below the button, showing:
    - A search field to filter agents by name
    - A compact **company filter** dropdown (visible only to admin users with access to multiple companies)
    - A scrollable list of agents, each displayed with its colored initial avatar and name
  </Step>
  
  <Step title="Search and Filter">
    - Type in the search field to instantly filter the agent list by name — results update as you type, with no need to press Enter
    - Admin users can narrow results further by selecting a specific company from the compact company dropdown at the top of the panel
    - Combine both search and company filter for precise results when managing agents across multiple organizations
  </Step>
  
  <Step title="Switch Agents">
    Click on any agent in the list to load it into the editor:
    - If you have unsaved changes on the current agent, a confirmation prompt appears asking you to **save**, **discard**, or **cancel** the switch
    - Once resolved, the selected agent loads immediately with its full configuration
    - Your editor layout, open tabs, and view preferences (minimap, focus mode, etc.) are preserved across the switch
  </Step>
</Steps>

<Info>
  The quick agent selector is only available while editing an existing agent (**edit mode**) and only on desktop screens. It is not shown while creating a brand-new agent from scratch, and on mobile you'll use the **Back to Agents** button instead.
</Info>

<Tip>
  Use **Cmd/Ctrl + Shift + A** as a keyboard shortcut to instantly open the agent selector without touching your mouse. Press it again, or click outside the dropdown, to close it.
</Tip>

## Archive Agents

<Info>
  Archiving is the recommended way to retire an agent you no longer need while keeping its configuration, call history, and analytics intact for future reference.
</Info>

Unlike deleting an agent, **archiving** deactivates it without permanently erasing its data. Archived agents stop receiving and making calls, but remain visible under the **Archived** filter for auditing or reactivation later.

### Archive an Agent

<Steps>
  <Step title="Open the Archive Option">
    From the Advanced Editor header toolbar or the agent details panel, click **Archive Agent**. This opens the **Archive Agent** modal.
  </Step>
  
  <Step title="Review the Confirmation">
    The modal summarizes exactly what will happen:
    - The agent will stop handling inbound and outbound calls immediately
    - All configuration, prompts, Knowledge Base, and call history are preserved
    - The agent can be restored from the **Archived** filter at any time
  </Step>
  
  <Step title="Choose Phone Number Handling">
    If the agent has an assigned phone number, the modal displays a dedicated **Release phone number** checkbox:
    - **Checked**: The associated phone number is immediately freed and returned to your available number pool, ready to be assigned to another agent
    - **Unchecked (default)**: The number stays reserved for this agent so it's automatically available again if you restore it later
    
    If the agent has no phone number assigned, this option is not shown.
  </Step>
  
  <Step title="Complete the Archive">
    Click the **Archive Agent** button to confirm. The button shows a loading state while the request is processed, and the agent is immediately deactivated and moved to the archived list once complete.
  </Step>
</Steps>

<Warning>
  Releasing the phone number is **irreversible** — once released, the number may be reassigned to a different agent or company. If you plan to reactivate this agent later with the same number, leave the **Release phone number** checkbox disabled.
</Warning>

### What Happens When You Archive

| Aspect | Behavior |
|--------|----------|
| **Call handling** | Immediately stops accepting inbound and outbound calls |
| **Configuration** | Fully preserved (prompts, tools, Knowledge Base, versions) |
| **Call history** | Retained and accessible in analytics |
| **Phone number** | Released only if the **Release phone number** checkbox was selected during archiving |
| **Visibility** | Moved out of the active agents list; accessible via the **Archived** filter |
| **Billing** | No further usage charges accrue for an archived agent |

### Restoring an Archived Agent

Archived agents can be reactivated at any time:

<Steps>
  <Step title="Locate the Agent">
    Use the **Archived** status filter in the agents list to find the agent you want to restore.
  </Step>
  <Step title="Restore">
    Click **Restore Agent** from the agent's details panel or quick actions menu.
  </Step>
  <Step title="Reassign a Phone Number (if needed)">
    If the original phone number was released during archiving, you'll need to assign a new number from the [Phone Numbers](/agents/phone-numbers) section before the agent can resume handling calls.
  </Step>
</Steps>

<Info>
  Restoring an agent does not automatically restore a released phone number. Make sure to assign a new number if the previous one was freed.
</Info>

## Advanced Agent Cloning

<Warning>
  Advanced cloning is available for admin users and agents with Advanced Editor enabled.
</Warning>

### Clone Agent with Enhanced Options

<Steps>
  <Step title="Select Agent">
    Choose the agent you want to clone from the agent list
  </Step>
  
  <Step title="Configure Clone">
    In the clone modal, specify:
    - **New agent name**
    - **Target company** (if you have access to multiple)
    - **Phone number assignment**
    - **Configuration mode**:
      - **Exact Copy**: Duplicate all settings
      - **Template Mode**: Copy structure, clear data
      - **Cross-Company**: Adapt for different organization
  </Step>
  
  <Step title="Advanced Settings">
    Choose what to clone:
    - ✅ Basic configuration and greetings
    - ✅ System prompts and instructions  
    - ✅ Knowledge Base documents
    - ✅ Actions & Tools configuration
    - ✅ Output parameters and formatting
    - ⚠️ Call history (optional)
    - ⚠️ Performance metrics (optional)
  </Step>
</Steps>

| **A/B Testing** | Create variants with different prompts |

- **Quick Access**: Click the agent name and avatar in the header toolbar to open the selector
