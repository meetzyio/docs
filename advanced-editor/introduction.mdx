## What is the Advanced Editor?

The **Advanced Editor** is Meetzy's comprehensive agent configuration tool that gives you full control over every aspect of your AI agent's behavior. It provides a professional-grade interface for creating, testing, and deploying conversational AI agents.

<Info>
The Advanced Editor is the recommended tool for creating production-ready agents with custom behaviors, integrations, and advanced logic.
</Info>

## Key Features

### Version Control System

The Advanced Editor includes a powerful version control system with:

- **Branching**: Create separate branches for feature development
- **Change tracking**: Monitor all modifications in real-time
- **Collaboration**: See when teammates are editing and avoid conflicts
- **Rollback**: Easily revert to previous versions if needed
- **Merge capabilities**: Combine changes from different branches

<Warning>
When multiple users edit the same agent simultaneously, the system will alert you to potential conflicts and help resolve them before publishing.
</Warning>

### Real-time Collaboration

Work together seamlessly with your team:

- Live cursors show where teammates are editing
- Automatic conflict detection and resolution
- Branch-based workflow for parallel development
- Comprehensive change history and audit trail

## Workflow Best Practices

<Steps>
  <Step title="Configure Basic Settings">
    Start with name, language, voice selection, and LLM model choice.
  </Step>
  <Step title="Set Up Greetings">
    Define how your agent introduces itself for different call types.
  </Step>
  <Step title="Write System Prompt">
    Describe your agent's role, personality, and conversation guidelines.
  </Step>
  <Step title="Add Knowledge Base">
    Upload documents and FAQs the agent can reference.
  </Step>
  <Step title="Configure Actions">
    Set up call transfers, webhooks, and other integrations.
  </Step>
  <Step title="Define Output">
    Specify what data to extract from each call.
  </Step>
  <Step title="Test in Playground">
    Validate agent behavior before publishing.
  </Step>
  <Step title="Review Changes">
    Use the Changes panel to review all modifications before committing.
  </Step>
  <Step title="Publish">
    Deploy your agent to production with confidence.
  </Step>
</Steps>

## Key Concepts

<AccordionGroup>
  <Accordion title="Versions & Branches" icon="code-branch">
    The Advanced Editor supports Git-like version control with branches for parallel development, comprehensive change tracking, and easy rollback capabilities. Create feature branches to work on improvements without affecting the live agent.
  </Accordion>
  <Accordion title="Live vs Development" icon="rocket">
    You can work on changes in development mode and only push them to production (Live) when you're satisfied with the results. The version control system ensures safe deployments.
  </Accordion>
  <Accordion title="Real-time Testing" icon="play">
    The Playground allows you to test your agent instantly without making actual phone calls, saving time and money.
  </Accordion>
  <Accordion title="Team Collaboration" icon="users">
    Multiple team members can work on the same agent simultaneously with real-time conflict detection and collaborative editing features.
  </Accordion>
</AccordionGroup>
