The Billy consumption panel provides comprehensive real-time visibility into your billing consumption with advanced filtering capabilities and administrative insights. This floating panel appears in the bottom-left corner of your dashboard when billing rules are configured.

<Info>
The Billy panel automatically appears when you have billing rules configured for your account. If you don't see it, contact your administrator to set up billing configuration.
</Info>

## Panel Overview

The enhanced Billy consumption panel displays:

- **Real-time consumption data** with intelligent period detection
- **Advanced filtering system** with predefined periods and custom date ranges
- **Cost breakdown by assistant** with detailed unit metrics
- **Administrative historical charts** for trend analysis
- **Billing adjustments** including minimums and corrections
- **Flexible view modes** for different user types

## Panel Interface

### Header Controls

The panel header includes:
- **Status indicator**: Green pulse for active monitoring, amber for errors
- **Refresh button**: Manual data refresh with loading state
- **Collapse/expand toggle**: Minimizes panel while keeping it accessible

### Expanding and Collapsing

<Steps>
<Step title="Locate the Panel">
The panel appears as a small indicator in the bottom-left corner with status-based color coding
</Step>

<Step title="Expand the Panel">
Click the expand button (down arrow) to view detailed consumption data and controls
</Step>

<Step title="Collapse When Needed">
Click the collapse button (up arrow) to minimize the panel. Your preference is automatically saved
</Step>
</Steps>

<Info>
The panel automatically saves your collapsed/expanded preference, restoring it when you return to the dashboard.
</Info>

## Advanced Filtering System

### Predefined Period Filters

The panel automatically generates billing periods based on your account configuration:

- **Current billing period** (default selection)
- **Previous billing periods** (up to 8 historical periods)
- **Automatically calculated periods** based on your billing cycle start day

<Steps>
<Step title="Open Filter Section">
Click "Filter" to expand the filtering controls
</Step>

<Step title="Select Predefined Period">
Choose from the dropdown of automatically generated billing periods
</Step>

<Step title="View Updated Data">
Data automatically refreshes when a new period is selected
</Step>
</Steps>

### Custom Date Range Filtering

For flexible analysis beyond predefined periods:

<Steps>
<Step title="Set Start Date">
Use the "From" date picker to select your analysis start date
</Step>

<Step title="Set End Date">
Use the "To" date picker to define the end of your analysis period
</Step>

<Step title="Apply Custom Range">
Click "Apply" to load consumption data for your custom date range
</Step>
</Steps>

<Warning>
Custom date ranges must have a start date before or equal to the end date. The panel will show validation errors for invalid date combinations.
</Warning>

### Filter Collapse and Expansion

The filter section can be collapsed to save space while still providing quick access to filtering options:

<Steps>
<Step title="Toggle Filter Visibility">
Click the "Filter" header to expand or collapse the filtering controls
</Step>

<Step title="Access Quick Filters">
When collapsed, the section still shows the current filter status
</Step>

<Step title="Expand for Full Control">
Open the section to access all filtering options including period selection and custom dates
</Step>
</Steps>

### Filter States and Error Handling

The filtering system includes comprehensive validation:

- **Date validation**: Prevents invalid date combinations
- **Loading states**: Shows "Recalculating consumption..." during data refresh
- **Error recovery**: Displays helpful error messages and retry options
- **Auto-synchronization**: Keeps filter state in sync with consumption data

## Understanding Consumption Data

### Breakdown by Assistant

Each assistant entry shows:

- **Assistant name** and identification
- **Usage metrics** with proper unit labels (minutes, calls, etc.)
- **Unit pricing** and subtotal calculations
- **Billing type** information

### Cost Calculation Structure

The panel displays a transparent cost breakdown:

```
Subtotal (all assistant usage)
+ Minimum Applied (billing minimums)
- Correction Amount (adjustments/credits)
= Total Cost
```

### Unit Display and Formatting

- **Automatic pluralization**: Units display correctly for singular/plural amounts
- **Localized formatting**: Numbers and currency display according to your locale
- **Consistent precision**: Monetary values show appropriate decimal places

## Data Refresh and States

### Manual Refresh

<Steps>
<Step title="Trigger Refresh">
Click the circular arrow refresh icon in the panel header
</Step>

<Step title="Monitor Progress">
The icon shows a spinning animation during data loading
</Step>

<Step title="Handle Errors">
If refresh fails, the status indicator changes to amber and error messages appear
</Step>
</Steps>

### Smart Refresh Logic

The refresh system intelligently handles different scenarios:

- **Period filters**: Refreshes data for the currently selected period
- **Custom ranges**: Re-applies custom date filters
- **Default mode**: Falls back to standard consumption loading
- **State preservation**: Maintains current filter selections

### Loading States

The panel provides clear feedback during operations:

- **"Recalculating consumption..."**: Data is being refreshed
- **"Loading billing history..."**: Historical chart data is loading
- **Disabled controls**: Prevent multiple simultaneous requests

### Error Recovery

When errors occur, the panel:
- Changes status indicator to amber
- Displays clear error messages
- Allows retry through manual refresh
- Maintains last successful data when possible

## Troubleshooting

<Tabs>
<Tab title="Panel Not Visible">
**Possible causes:**
- No Billy billing profile configured
- Insufficient permissions for consumption viewing
- Billing rules not set up for your account

**Solutions:**
- Contact your administrator to configure Billy billing
- Verify your account has consumption viewing permissions
- Check if billing rules exist for your assistants
</Tab>

<Tab title="Filters Not Working">
**Possible causes:**
- Invalid date range selections
- Network connectivity issues
- Server-side date processing errors

**Solutions:**
- Verify start date is before or equal to end date
- Try using predefined periods instead of custom ranges
- Check your internet connection and retry
</Tab>

<Tab title="Historical Chart Missing">
**Possible causes:**
- Non-admin account (feature restricted)
- Insufficient billing history data
- Chart rendering failures

**Solutions:**
- Verify you have administrative permissions
- Contact support if you should have admin access
- Try refreshing the panel to reload chart data
</Tab>

<Tab title="Incorrect Consumption Data">
**Possible causes:**
- Cached data showing outdated information
- Recent billing changes not yet reflected
- Time zone calculation differences

**Solutions:**
- Use the manual refresh button
- Clear browser cache and reload
- Verify your account time zone settings
</Tab>
</Tabs>
