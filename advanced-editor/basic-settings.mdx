## Overview

Basic Settings is the first section in the Advanced Editor where you configure the fundamental properties of your AI agent. These settings define how your agent sounds, which language it speaks, which AI model powers its intelligence, and the foundational components that shape its behavior.

## Agent Identification

### Agent Name

Give your agent a descriptive name that helps you identify it in your dashboard.

<Info>
Use clear naming conventions like "Customer Support - English" or "Sales Outbound - Spanish" to easily manage multiple agents.
</Info>

### Company Selection (Admin Only)

Administrators can assign agents to specific companies in the system.

<Warning>
Company assignment affects which resources and phone numbers are available to the agent.
</Warning>

## Twilio Account Management (Admin Only)

<Tabs>
  <Tab title="Account Selection">
    Administrators can select which Twilio account to use for new agents:

    <Steps>
      <Step title="Choose Twilio Account">
        Select from available Twilio accounts in the dropdown
      </Step>
      <Step title="Phone Number Assignment">
        Once an account is selected, available phone numbers for that account will be loaded
      </Step>
      <Step title="Automatic Configuration">
        The agent will be configured with the selected Twilio account credentials
      </Step>
    </Steps>
  </Tab>
  <Tab title="Account Switching">
    <Warning>
    Changing Twilio accounts will reload the available phone numbers list and may affect call routing.
    </Warning>
  </Tab>
</Tabs>

## Phone Configuration

### Phone Numbers

<Tabs>
  <Tab title="New Agents">
    When creating a new agent, select from available phone numbers in your Twilio account:

    <Steps>
      <Step title="Select Twilio Account">
        Administrators can choose which Twilio account to use for the agent
      </Step>
      <Step title="Choose Phone Number">
        Pick an available phone number from the dropdown list
      </Step>
      <Step title="Request More Numbers">
        Use the "Request more numbers" link if you need additional phone numbers from Twilio
      </Step>
    </Steps>

    <Info>
    Phone numbers are filtered by the selected Twilio account. Make sure to select the correct account before choosing a phone number.
    </Info>
  </Tab>
  <Tab title="Existing Agents">
    Phone numbers for existing agents are read-only and cannot be changed after creation.

    <Warning>
    Once an agent is created, the phone number assignment is permanent and cannot be modified.
    </Warning>
  </Tab>
</Tabs>

### Caller IDs (Client Numbers)

Configure client phone numbers for outbound calls with comprehensive management features:

<Tabs>
  <Tab title="Adding Caller IDs">
    <Steps>
      <Step title="Add New Caller ID">
        Click "Add Caller ID" to open the configuration modal
      </Step>
      <Step title="Enter Phone Details">
        Provide a name and phone number for identification
      </Step>
      <Step title="Twilio Verification">
        Twilio will initiate verification by calling the provided number
      </Step>
      <Step title="Enter Verification Code">
        Input the verification code received during the call
      </Step>
      <Step title="Complete Setup">
        The caller ID will be available for outbound calls once verified
      </Step>
    </Steps>
  </Tab>
  <Tab title="Managing Caller IDs">
    **Status Management:**
    - **Verified**: Ready to use for outbound calls
    - **Pending**: Waiting for verification code
    - **Failed**: Verification failed, requires retry

    **Available Actions:**
    - **Check Status**: Verify current validation status with Twilio
    - **Update Status**: Refresh the verification status
    - **Retry Verification**: Restart the verification process
    - **Delete**: Remove caller ID from the agent
  </Tab>
  <Tab title="Verification Process">
    <Info>
    **Automatic Calling**: When you add a caller ID, Twilio automatically calls the number within 30 seconds to provide the verification code.
    </Info>

    <Info>
    **Code Validity**: Verification codes are valid for a limited time. Complete the verification process promptly.
    </Info>

    <Warning>
    Only verified caller IDs can be used for outbound calls. Unverified numbers will not appear in your outbound calling options.
    </Warning>
  </Tab>
</Tabs>

### Advanced Voice Selection

Configure your agent's voice characteristics with enhanced options:

<Tabs>
  <Tab title="Voice Library">
    Browse and select from an extensive collection of pre-configured voices:
    
    **Filter Options:**
    - **All Voices**: Complete voice library
    - **Male**: Male voice options
    - **Female**: Female voice options
    - **Search**: Find specific voices by name or characteristics

    **Voice Preview:**
    - Click the play button to hear voice samples
    - Test different voices before selection
    - Ensure voice matches your brand and use case
  </Tab>
  <Tab title="Voice Agents">
    Choose from specialized voice agents with unique personalities and speaking styles:

    <Steps>
      <Step title="Browse Voice Agents">
        View available voice agents with different specialties and characteristics
      </Step>
      <Step title="Filter by Specialty">
        Filter agents by specialty (Customer Service, Sales, Support, etc.)
      </Step>
      <Step title="Preview Personality">
        Listen to samples to understand each agent's unique speaking style
      </Step>
      <Step title="Select and Apply">
        Choose a voice agent that matches your use case and brand voice
      </Step>
    </Steps>

    **Available Specialties:**
    - Customer Service Representatives
    - Sales Professionals
    - Technical Support Agents
    - Appointment Schedulers
    - General Purpose Assistants
  </Tab>
  <Tab title="Voice Tuning">
    Fine-tune voice parameters for optimal performance:
    
    | Parameter | Range | Description | Recommended |
    |-----------|-------|-------------|-------------|
    | Speed | 0.70x - 1.20x | Controls speaking pace | 1.00x - 1.10x |
    | Stability | 0.0 - 1.0 | Voice consistency across utterances | 0.1 - 0.3 |
    | Similarity Boost | 0.0 - 1.0 | Voice naturalness and clarity | 0.6 - 0.8 |

    <Info>
    **Default Settings**: Speed: 1.06x, Stability: 0.15, Similarity Boost: 0.65 provide optimal balance for phone conversations.
    </Info>
  </Tab>
</Tabs>

## Troubleshooting & Common Issues

<Tabs>
  <Tab title="Phone Configuration">
    <Warning>
    **Phone Number Assignment**: Phone numbers cannot be changed after agent creation. Plan your phone number allocation carefully and verify availability before creating agents.
    </Warning>

    <Warning>
    **Twilio Account Access**: Ensure your Twilio account has sufficient credits and proper configuration for the regions you plan to call.
    </Warning>
  </Tab>
  <Tab title="Caller ID Issues">
    **Common Caller ID Problems:**
    - Verification calls not received within 2 minutes
    - Invalid verification codes
    - Numbers showing as unverified after code entry

    **Solutions:**
    - Check phone number format (+1XXXXXXXXXX for US numbers)
    - Ensure the phone can receive calls during verification
    - Wait up to 2 minutes for verification calls
    - Use the "Retry Verification" option if needed
    - Contact support for persistent verification issues
  </Tab>
  <Tab title="Model & Voice">
    <Warning>
    **Model Changes**: Changing the AI model may significantly affect your agent's behavior patterns. Always test thoroughly in the Playground after switching models.
    </Warning>

    <Warning>
    **Voice Language Compatibility**: Ensure your selected voice supports your chosen language for optimal pronunciation and performance.
    </Warning>
  </Tab>
</Tabs>
